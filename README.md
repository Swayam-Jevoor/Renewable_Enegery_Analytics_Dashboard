# Renewable Energy Performance Dashboard

## Overview
This Power BI dashboard presents key metrics and comparisons for different renewable energy sources, focusing on production, consumption, storage, and efficiency.

## Key Metrics
- Total Energy Consumption: 3.39bn MWh
- Total Installed Capacity: 7.44M MW
- Total Energy Production: 3.79M GWh
- Energy Storage Capacity: 75.45K GWh
- Average Storage Efficiency: 75.22%

## Visualizations
- **Energy Production vs Consumption** – Comparison across energy types  
- **Energy Production per Type** – Distribution of production levels  
- **Average Storage Efficiency** – Efficiency comparison by source  
- **Production Share (Donut Chart)** – Contribution by each energy type  

## Filters
- Type of Renewable Energy  
- Grid Integration Level  
- Storage Efficiency (%)  
- Initial Investment (USD)  
- Funding Sources  

## Tools Used
- Power BI  
- DAX  
- Data Modeling  

## Usage
Open the `.pbix` file in Power BI Desktop and use the filters to explore the data.

## Note
This dashboard is descriptive and focuses on comparison rather than trend or predictive analysis.
